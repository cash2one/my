学校概况

美学堂整体造型艺术教育机构是由从事化妆造型、美容专业教育十多年的专家创建。教育机构位于文化氛围一流的文化路上，时尚品位的教学环境。所有师资都获得SIDESCO、ITEC等国际权威机构认证的证书，并有十年以上的教学经验！

随着人们生活条件的提高，越来越多的人们重视自身的形象，为了打造专业的人才，美学堂汇集国际最新的咨询和与专业实用技术，运用国际的教学方式，引进欧美、日韩等流行趋势，并有效结合中国民族文化，走在时尚前沿！使学员更快的掌握实用技术和市场的结合，成为现今时尚的佼佼者!

学员在学习过程中，学会用不同的的视觉审视美、认识美、创造美，开创美学的最高境界！并同时提高学员的审美品味，打造专业形象。从实际的技术入手，切实结合市场竞争和时尚潮流动态，打造实用性、竞争性的人才！

本机构集各类商业演出、专业化妆造型、新娘顾问及婚庆策划、个人形象顾问、彩妆师和化妆讲师的培训、学生就业推广、开设个人工作形象室的营销和策划及店面管理与整合。
